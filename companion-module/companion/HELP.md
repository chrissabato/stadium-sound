# Stadium Sound

Enable **OSC & iPad Remote** in Stadium Sound's Settings, then enter the computer's IP address and OSC UDP port here (default `9000`).

The module provides actions for track playback by ID, bank selection by ID/name/one-based number, stop, fade, random, and master volume. Feedback can show the active track or bank. Track and bank IDs are stable within an event set and are visible in the iPad remote's browser data or event-set file.
