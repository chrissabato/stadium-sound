Features

Built for the booth,
not the bedroom.

⚡
Instant Playback

Files are pre-decoded at load time. Click a track and it plays — no buffer spin-up, no wait.

☰
Named Banks

Organize tracks into banks by category — walk-ons, intros, timeouts, highlights. Switch banks mid-event without losing your place.

▶
Playlists

Queue tracks from any bank into an ordered playlist and step through your run of show in sequence instead of hunting across grids.

◎
Monitor Mode

Route audio to a private headphone output to preview a track before playing it to the venue.

⊞
Track Editor

Waveform view with draggable in/out points. Trim intros and set player name fields for walk-on cues.

⌨
Keyboard Shortcuts

Assign a hotkey to any track for instant, no-mouse triggering. Pull up the full cheat sheet any time from the toolbar.

⇄
Sports Sounds Pro Import

Already running Sports Sounds Pro? Import your existing .set files directly — banks, track names, and file paths all carry over. Use File → Import Sports Sounds Pro Set… to get started.
