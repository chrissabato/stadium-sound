Features

Built for the booth,
not the bedroom.

⚡
Instant Playback

Files are pre-decoded at load time. Click a track and it plays — no buffer spin-up, no wait.

☰
Named Banks

Organize tracks into banks by category — walk-ons, intros, timeouts, highlights. Switch banks mid-event without losing your place.

◑
Fade Controls

Per-event fade in, fade out, and crossfade settings. Transitions that sound like a broadcast, not a laptop.

◎
Monitor Mode

Route audio to a private headphone output to preview a track before playing it to the venue.

⊞
Track Editor

Waveform view with draggable in/out points. Trim intros, cut silence, set player name fields for walk-on cues.

↺
Auto-Update

New versions download in the background and install on next restart. Always current, zero effort.
